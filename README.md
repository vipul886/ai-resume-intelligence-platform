# 🚀 AI Resume Intelligence Platform

An AI-powered Resume Analyzer and Job Matching system developed using **Python, FastAPI, NLP and Machine Learning**.

This project simulates a real-world intelligent recruitment assistant that helps evaluate resumes, calculate ATS scores and match candidates with job descriptions.

---

## ⭐ Key Features

✅ Resume skill extraction using NLP
✅ ATS compatibility scoring
✅ Resume vs Job Description similarity matching
✅ Skill gap analysis
✅ AI-based recommendations

---

## 🧠 Tech Stack

* Python
* FastAPI
* Natural Language Processing (NLP)
* Scikit-learn
* Machine Learning
* REST API Development

---

## 🏗 Project Architecture

* **API Layer** → FastAPI endpoints
* **Processing Layer** → Resume parsing & NLP pipeline
* **ML Layer** → ATS scoring & job matching
* **Data Layer** → Job database & resume samples

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
uvicorn api.main:app --reload
```

Then open in browser:
👉 http://127.0.0.1:8000/docs
